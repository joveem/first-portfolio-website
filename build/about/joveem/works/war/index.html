participation in

War (by Grow Games)


Gameplay

Optimizations

I made some revisions to the game related to performance. For example: a new game map was having performance issues on mid-range devices. I managed to do an optimization where it went from 25 FPS to 60+ without sacrificing the original look

UI responsiveness issues fixing

I solved 99% of the responsiveness and "safe area" problems by making the content of all menus and scenes work on all screen formats and devices, solving problems with unexpected behavior on screens with unexpected formats and other problems, such as "camera cutouts" on some mobile devices that have the camera above the screen

General development tools

I made some tools related to the development itself, for example a tool that generated screenshot files in different resolutions to be used in some stores that requested screenshots in a large list of specific resolutions (yes, I'm talking about Apple haha)

Tournament

I led a team of one developer in implementing tournaments. This made it possible to define several tournaments in advance, being able to configure the announcement, start, end and end dates of the results and they took place without supervision or manual actions.

App versions handling

I implemented version handling, which distinguished application versions from users, where it was possible to give warnings of non-mandatory updates to specific user versions while blocking other versions with mandatory updates

Customer Service Center integration

I designed integration with the customer service center and led a team of one developer to implement it. Inthis integration, the user could get in touch within the game. This generated an email with the user's message, device information (such as operating system version, amount of RAM, screen resolution), player information, game logs and a screenshot. This was sent to the support team who contacted the user and created a problem report/ticket

Rewarded ads

I designed and implemented a solution for rewarded advertising. This allowed players to watch an advertisement to play a game when the power went out

Localization implementation

I designed a localization solution, and led a team of one more developer to implement it

Build automation

I architected and implemented an atomization of builds. It allowed reducing the time spent on configurations and monitoring manual builds, generating builds for different platforms (Android, iOS and WebGL) at once at the touch of a button

General Game Desing changes implementation

I implemented several small game design changes, most were related to visuals and user experience, such as changing the ways of interacting with the map, feedback and HUD

General bug fixes

I made several small corrections to the game, most of them were related to problems with the subscription, removing ads, adding and using energy, controlling the camera in the game, etc.
